## 本项目，你可以学到

* 如何将 JS SDK 用在前端，暴露了 AppId 或 Appkey 的时候，如何通过 ACL 权限控制，保护数据。
* 如何使用 AV.View 做交互多的界面
* 如何用 AV.Router，AV.history 来导航页面
* 如何将 Dom 的各种事件与 AV.Object的函数相绑定，来实现数据与页面的大量交互
